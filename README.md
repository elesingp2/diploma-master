# Расчетные ноутбуки диплома

Репозиторий содержит компактный публичный срез расчетных ноутбуков для диплома.
В нем оставлены только файлы, которые нужны, чтобы открыть расчеты на GitHub и
при необходимости воспроизвести их локально:

- `notebooks/` -- выполненные ноутбуки с сохраненными выводами для просмотра на GitHub;
- `src/thesis_modeling/` -- переиспользуемые функции, которые вызываются из ноутбуков;
- `data/genfoam/` -- компактные тепловые ряды GeN-Foam, используемые как входные данные;
- `pyproject.toml` -- зависимости Python и метаданные пакета.

## Воспроизведение

```bash
python3 -m venv .venv
. .venv/bin/activate
python -m pip install -e '.[chemistry,notebooks]'
python -m jupyter nbconvert --to notebook --execute --inplace notebooks/*.ipynb
```

Для равновесных оценок `H2` требуется Cantera. Сгенерированные рисунки, черновики
LaTeX, состояние агентов, локальные окружения и прочие процессные артефакты
исключены через `.gitignore`.
